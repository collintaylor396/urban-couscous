# Project Overview

A minimal static HTML redirect page. When visited, it immediately redirects the browser to a Dropbox Paper document ("ATTACHED DOCUMENTS").

## Stack

- Plain HTML (single `index.html`)
- No framework, no dependencies, no build step

## How to run

The app is served with Python's built-in HTTP server:

```
python3 -m http.server 5000
```

This is configured as the "Start application" workflow on port 5000.
